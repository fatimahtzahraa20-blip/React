# 🎬 Video Editor Portfolio

A professional Video Editor Portfolio built with **React, TypeScript, Vite, Tailwind CSS, and Supabase**. This project showcases video editing work, allows visitors to contact the editor, and provides an admin dashboard for managing videos, categories, and portfolio settings.

---

## ✨ Features

- Responsive modern UI
- Video showcase
- Video categories
- Contact form
- Admin login
- Dashboard
- Video management
- Category management
- Portfolio settings
- Supabase database integration
- Protected routes
- Form validation using Zod

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
